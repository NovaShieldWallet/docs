## Checking Transaction Status

Tap the **info icon** on the transaction details screen to see real-time status updates.

## Status Types

### Pending
Your transaction has been submitted to the network and is awaiting confirmation.

### Confirmed
Your transaction has been successfully processed and added to the blockchain.

### Failed
Your transaction was rejected by the network. Common reasons:
- Insufficient SOL for fees
- Slippage tolerance exceeded
- Token restrictions (honeypot)

## Transaction Details

View complete transaction information including:
- Transaction signature
- Block confirmation
- Fee amount
- Timestamp
- Token amounts transferred
