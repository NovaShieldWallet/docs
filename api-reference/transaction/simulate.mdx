---
title: 'Transaction Simulation'
openapi: 'POST /api/simulate-transaction'
---

Simulates a Solana transaction to preview effects before signing.

## Request

```json
{
  "transaction": "BASE64_ENCODED_TRANSACTION",
  "signerPublicKey": "7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU"
}
```

## Response

Returns simulation results including:
- Transaction success/failure prediction
- Program logs
- Risk warnings
- Account changes
- Recommended actions
