## AI-Powered Analysis

Every security check includes AI-enhanced risk analysis that goes beyond static databases:

- **Risk Scoring**: 0-100 risk score with detailed classification
- **Smart Recommendations**: Actionable security advice
- **Pattern Detection**: Identify threats that databases miss
- **Real-Time Intelligence**: Stay ahead of emerging scams

## Authentication

All API endpoints require Bearer token authentication.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

<Tip>
Contact NovaShield to get your API key and start protecting your users.
</Tip>

## Quick Start

<Steps>
<Step title="Get your API key">
  Contact the NovaShield team to receive your Bearer token.
</Step>

<Step title="Make your first request">
  ```bash
  curl -X POST https://secure.novashield.space/api/token/security \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "chainId": "1",
      "contractAddresses": ["0xdac17f958d2ee523a2206206994597c13d831ec7"]
    }'
  ```
</Step>

<Step title="Analyze the results">
  Review the security data and AI analysis to identify risks and protect your users.
</Step>
</Steps>
