## AI-Powered Analysis

Every security check includes intelligent risk assessment:

- **Risk Scores**: 0-100 risk score with severity levels (SAFE, LOW, MEDIUM, HIGH, CRITICAL)
- **Clear Verdicts**: Human-readable verdicts like "🚨 EXTREME RUG PULL RISK - DO NOT BUY!"
- **Actionable Recommendations**: Specific advice on how to protect users
- **Real-Time Intelligence**: Up-to-date threat data beyond static databases

## Quick Start

<Steps>
<Step title="Check a token for rug pull">
  ```bash
  curl -X POST https://secure.novashield.space/api/token/rug-check \
    -H "Content-Type: application/json" \
    -d '{
      "contractAddress": "HrPpKj6BPmCH4HmLiNCmJ7HiFmv8CYiM7qRmnqWipump"
    }'
  ```
</Step>

<Step title="Analyze the response">
  ```json
  {
    "success": true,
    "data": {
      "contractAddress": "HrPpKj6BPmCH4HmLiNCmJ7HiFmv8CYiM7qRmnqWipump",
      "chain": "solana",
      "rugPullRisk": {
        "score": 100,
        "level": "CRITICAL",
        "verdict": "🚨 EXTREME RUG PULL RISK - DO NOT BUY!",
        "canTrade": false,
        "isHoneypot": false
      },
      "tokenMetrics": {
        "holderCount": 5,
        "name": "Present Coin",
        "symbol": "Present",
        "liquidity": 0,
        "marketCap": 0
      }
    }
  }
  ```
</Step>

<Step title="Protect your users">
  Use the `verdict`, `canTrade`, and `riskLevel` fields to block dangerous transactions and warn users about risks.
</Step>
</Steps>
