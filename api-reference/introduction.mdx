## Local-First Detection

NovaShield uses a local-first approach:
- ✅ Primary analysis runs on our infrastructure
- ✅ Fast response times (typically <500ms)
- ✅ No external API dependencies for core checks
- ✅ Multiple security modules working together

## Risk Assessment

Every security check includes intelligent risk scoring:

- **Risk Scores**: 0-100 with severity levels (SAFE, LOW, MEDIUM, HIGH, CRITICAL)
- **Clear Verdicts**: Human-readable assessments
- **AI Analysis**: Comprehensive insights with key risks and recommendations
- **Trading Recommendations**: Boolean `canTrade` flags

## Quick Start

<Steps>
<Step title="Check token security">
  ```bash
  curl https://secure.novashield.space/token/security/HrPpKj6BPmCH4HmLiNCmJ7HiFmv8CYiM7qRmnqWipump
  ```
</Step>

<Step title="Analyze the response">
  ```json
  {
    "success": true,
    "data": {
      "overallScore": 15,
      "riskLevel": "CRITICAL",
      "verdict": "🚨 EXTREME RUG PULL RISK - DO NOT BUY!",
      "canTrade": false,

      "authorities": { "freezeAuthority": null, "mintAuthority": null },
      "liquidity": { "locked": false, "lpValue": 0 },
      "holders": { "top10Concentration": 95, "totalHolders": 5 },

    }
  }
  ```
</Step>

<Step title="Protect your users">
