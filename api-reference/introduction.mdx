## NovaShield APIs

**Security** - Detect rug pulls, honeypots, and phishing sites before your users get scammed.  
**Privacy Cash** - Send SOL with zero trace. Complete privacy for sensitive transfers.

Both available as APIs and integrated into the **[Nova iOS Wallet](/quickstart)**.

## Core Features

**🛡️ Token Security** - AI-powered analysis catches scams before they happen  
**🕵️ Honeypot Detection** - Verify tokens are actually tradeable  
**🔒 Privacy Cash** - Untraceable SOL transfers (1% + 0.006 SOL fee)  
**🎣 Phishing Protection** - Block typosquatting and fake sites  
**🔮 Transaction Simulation** - Preview what transactions do before signing

## Base URL

```
https://secure.novashield.space
```

## Example: Check Token Security

```bash
curl https://secure.novashield.space/token/security/MINT_ADDRESS
```

## Example: Send Private SOL

```bash
curl -X POST https://secure.novashield.space/privacy-cash/send \
  -H "Content-Type: application/json" \
  -d '{
    "fromPrivateKey": "[...]",
    "toAddress": "RECIPIENT",
    "amount": 0.05,
    "network": "mainnet-beta"
  }'
```

## Authentication

All endpoints are currently **open** - no API key required.
