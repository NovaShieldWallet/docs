## NovaShield Security API

Comprehensive Solana token security scanning with local-first detection. Analyze tokens for rug pulls, detect honeypots, and protect users from phishing sites.

Available as both an **API** for developers and integrated into the **[Nova iOS Wallet](/quickstart)** for end users.

## Authentication

API endpoints are currently **open** and do not require authentication.

## Base URL

```
https://secure.novashield.space
```

## Example Request

Check a token for security risks:

```bash
curl https://secure.novashield.space/token/security/HrPpKj6BPmCH4HmLiNCmJ7HiFmv8CYiM7qRmnqWipump
```

## Example Response

```json
{
  "success": true,
  "data": {
    "overallScore": 15,
    "riskLevel": "CRITICAL",
    "verdict": "🚨 EXTREME RUG PULL RISK - DO NOT BUY!",
    "canTrade": false,
    "honeypot": { "isHoneypot": false, "canBuy": true, "canSell": true },
    "authorities": { "freezeAuthority": null, "mintAuthority": null },
    "liquidity": { "locked": false, "lpValue": 0 },
    "holders": { "top10Concentration": 95, "totalHolders": 5 },
    "aiAnalysis": {
      "summary": "Critical risk token with extreme concentration",
      "keyRisks": ["Extremely low holder count", "No liquidity"],
      "recommendations": ["Do not trade this token"],
      "confidence": 99
    }
  }
}
```

## Key Response Fields

- **overallScore** - Security score (0-100, lower = more dangerous)
- **riskLevel** - SAFE, LOW, MEDIUM, HIGH, or CRITICAL
- **verdict** - Human-readable assessment
- **canTrade** - Boolean recommendation
- **aiAnalysis** - AI-powered insights with key risks and recommendations
