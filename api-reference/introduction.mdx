## What is NovaShield?

NovaShield is an on-chain security engine for Solana that uses AI and threat intelligence to analyze transactions, tokens, addresses, dApps, and URLs — helping you predict and block threats before users click Sign.

## Base URLs

```
Production:  https://api.novashield.so
Staging:     https://sapi.novashield.so
```

## Versioning

All endpoints are versioned under `/v1/...`.

## API Categories

- **Scanner** — Scan transactions before signing, simulate effects, check address reputation, analyze dApps, and detect phishing URLs.
- **Token Security** — Full token security analysis, bundler detection, and rug-style trader summaries.
- **System** — Health and info endpoints.

## Authentication

No API key required at this time. Rate limiting and abuse protections may apply.

## Responses and Errors

All endpoints return consistent JSON with machine-readable codes. See Error Codes.
