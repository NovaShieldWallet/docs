---
title: 'Honeypot Detection'
openapi: 'GET /token/honeypot/{mint}'
---

Detects if a token is a honeypot (can buy but cannot sell).

## Parameters

- **mint** (path, required) - Token mint address

## Response

```json
{
  "success": true,
  "data": {
    "isHoneypot": false,
    "canBuy": true,
    "canSell": true,
    "confidence": 100,
    "reasons": ["Buy and sell operations successful"]
  }
}
```

## Fields

- **isHoneypot** - `true` if honeypot, `false` if safe, `null` if unable to verify
- **canBuy** - Can execute buy operations (or `null`)
- **canSell** - Can execute sell operations (or `null`)
- **confidence** - 0 = unable to verify, 100 = certain
- **reasons** - Explanations for the result

## Interpreting Results

| Result | Verdict |
|--------|---------|
| `isHoneypot: false` | Safe to trade |
| `isHoneypot: true` | HONEYPOT - Do not buy |
| `isHoneypot: null` | Unable to verify |
