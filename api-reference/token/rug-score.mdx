---
title: 'Rug Pull Probability Score'
openapi: 'GET /token/rug-score/{mint}'
---

Fast rug pull risk assessment for a Solana token.

## Parameters

- **mint** (path, required) - Token mint address

## Response

```json
{
  "success": true,
  "data": {
    "score": 25,
    "level": "LOW",
    "verdict": "Low rug pull risk detected",
    "canTrade": true
  }
}
```

## Score Ranges

- **0-20**: SAFE
- **21-40**: LOW risk
- **41-60**: MEDIUM risk
- **61-80**: HIGH risk
- **81-100**: CRITICAL risk
