## Overview

Detects coordinated buying patterns that indicate bundler activity or wash trading. This advanced analysis identifies groups of wallets that appear to be controlled by the same entity.

## What Are Bundler Clusters?

Bundler clusters are groups of wallets that exhibit coordinated behavior:
- Buy at the same time (within seconds)
- Use the same funding source
- Have similar transaction patterns
- Execute trades in sequence

This is a sophisticated scam technique used to:
- Fake organic demand
- Control price action
- Set up pump & dumps
- Appear legitimate to automated tools

## Analysis Periods

The endpoint supports two analysis modes via the `period` query parameter:

### Launch Period (`period=launch`)
Analyzes the first 100-500 transactions after token creation:
- Detects coordinated sniping
- Identifies initial distribution manipulation
- Catches bundler wallets at creation

### Recent Period (`period=recent`)
Analyzes the last 1000 transactions:
- Detects ongoing manipulation
- Identifies active trading clusters
- Monitors for dump coordination

## Detection Signals

The analysis looks for multiple coordinated patterns:

### Temporal Clustering
- Multiple buys within 1-5 second windows
- Synchronized transaction batches
- Time-based correlations

### Funding Analysis
- Common funding wallets
- Sequential wallet creation
- Identical SOL amounts deposited

### Trading Patterns
- Similar buy amounts
- Matching slippage settings
- Identical DEX routing

## Response Insights

Typical response includes:
- **Cluster count** - Number of detected coordinated groups
- **Cluster sizes** - How many wallets in each group
- **Risk score** - Overall bundling risk assessment
- **Wallet addresses** - Identified coordinated wallets
- **Confidence level** - Detection certainty

## Risk Assessment

| Clusters | Size | Risk Level |
|----------|------|------------|
| 0-1 | Any | LOW ✅ |
| 2-3 | <5 wallets | MEDIUM ⚠️ |
| 2-3 | 5+ wallets | HIGH 🚨 |
| 4+ | Any | CRITICAL 🚨 |

## Use Cases

- **Launch monitoring** - Check new tokens for coordinated sniping
- **Market analysis** - Identify artificial volume
- **Scam detection** - Flag sophisticated bundling operations
- **Trading bots** - Avoid manipulated markets

## Best Practices

When bundler clusters are detected:
1. 🛑 **HIGH/CRITICAL** - Do not trade
2. ⚠️ **MEDIUM** - Investigate further before trading
3. ✅ **LOW** - Proceed with normal caution

Always combine this check with:
- Holder concentration analysis (`/token/bundled/{mint}`)
- LP analysis (`/token/lp/{mint}`)
- Overall security check (`/token/security/{mint}`)
