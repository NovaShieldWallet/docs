---
title: 'Complete Security Analysis'
openapi: 'GET /token/security/{mint}'
---

Runs multiple security modules and returns a comprehensive security report for a Solana token.

## Parameters

- **mint** (path, required) - Token mint address
- **deployer** (query, optional) - Deployer address for enhanced analysis

## Response

```json
{
  "success": true,
  "data": {
    "overallScore": 85,
    "riskLevel": "LOW",
    "verdict": "Token appears safe with minor risks",
    "canTrade": true,
    "honeypot": { "isHoneypot": false, "canBuy": true, "canSell": true },
    "authorities": { "freezeAuthority": null, "mintAuthority": null },
    "liquidity": { "locked": true, "lpValue": 150000 },
    "holders": { "top10Concentration": 25, "totalHolders": 1250 },
    "aiAnalysis": {
      "summary": "Token shows healthy distribution and locked liquidity",
      "keyRisks": ["Minor: Recent launch"],
      "recommendations": ["Verify team before large investments"],
      "confidence": 95
    }
  }
}
```

## Key Fields

- **overallScore** - Security score (0-100, higher = safer)
- **riskLevel** - SAFE | LOW | MEDIUM | HIGH | CRITICAL
- **canTrade** - Boolean trading recommendation
- **aiAnalysis** - AI-powered insights with risks and recommendations
