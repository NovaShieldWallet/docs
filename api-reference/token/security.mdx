## Overview

This endpoint runs all 10 security modules and returns a comprehensive security report for a Solana token. It provides an overall risk assessment, AI-powered analysis, and detailed insights into:

- Honeypot detection
- Authority analysis (freeze/mint)
- Liquidity pool analysis
- Holder concentration
- And 6 additional security checks

## Use Cases

- **Pre-trade due diligence** - Check a token before buying
- **Portfolio monitoring** - Regularly scan tokens you hold
- **Integration** - Add security checks to your dApp or wallet

## Parameters

The endpoint accepts the token mint address as a path parameter, with an optional `deployer` query parameter for enhanced analysis.

## Response Fields

### Overall Assessment
- **overallScore** - Composite security score (0-100, higher is better)
- **riskLevel** - Classification: SAFE, LOW, MEDIUM, HIGH, or CRITICAL
- **verdict** - Human-readable risk assessment
- **canTrade** - Boolean recommendation on whether trading is advisable

### AI Analysis
The `aiAnalysis` object provides:
- **summary** - Natural language summary of findings
- **keyRisks** - Array of identified risks
- **recommendations** - Actionable advice
- **confidence** - AI confidence level (0-100)

### Security Modules
Detailed results from each of the 10 security modules:
- `honeypot` - Buy/sell restriction detection
- `authorities` - Freeze and mint authority analysis
- `liquidity` - LP lock status and depth
- `holders` - Distribution and concentration metrics

## Rate Limits

This endpoint is rate limited. See the `429` response for details on limits and retry windows.
