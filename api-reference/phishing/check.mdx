---
title: 'Phishing Detection'
openapi: 'POST /api/phishing/check'
---

Detects phishing websites and typosquatting attempts.

## Request

```json
{
  "url": "https://example.com"
}
```

## Response

```json
{
  "success": true,
  "data": {
    "url": "https://cpump.fun",
    "isPhishing": true,
    "confidence": 95,
    "riskLevel": "CRITICAL",
    "verdict": "PHISHING SITE DETECTED - DO NOT CONNECT!",
    "similarTo": "pump.fun"
  }
}
```

## Fields

- **isPhishing** - Whether the URL is a phishing site
- **confidence** - Detection confidence (0-100)
- **riskLevel** - SAFE | LOW | MEDIUM | HIGH | CRITICAL
- **similarTo** - Legitimate domain being impersonated (if applicable)
