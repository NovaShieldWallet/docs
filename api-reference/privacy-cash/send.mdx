openapi: 'POST /privacy-cash/send'

**SOL ONLY** - No tokens supported. Send SOL with complete privacy.

## How It Works

1. Call endpoint with amount and recipient
2. Receive secure address
3. Send SOL to that address (you keep your private key!)
4. Transfer auto-executes when funded (checks every 2s)
5. Address expires in 30 minutes

## Request

```json
{
  "amount": 0.05,
  "toAddress": "RECIPIENT_ADDRESS",
  "network": "mainnet-beta"
}
```

**You NEVER send your private key!**

## Response

```json
{
  "success": true,
  "data": {
    "sendTo": "SECURE_ADDRESS",
    "transferId": "transfer-123",
    "expiresAt": 1700001800
  },
  "message": "Send SOL to 'sendTo' address. Transfer executes automatically when funded."
}
```

## Next Steps

1. Send SOL to the `sendTo` address
2. Use `transferId` to check status
3. Transfer completes automatically

## Fees & Minimums

- **Minimum**: 0.04 SOL (recipient gets 0.03 after fees)
- **Total fee**: 1% + 0.006 SOL
- **Expiry**: 30 minutes

## Important

⚠️ **SOL ONLY** - Do not send tokens to address  
⚠️ **30 minute expiry** - Funds returned if not used  
✅ **No private key needed** - You remain in control
