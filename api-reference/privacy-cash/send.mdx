title: 'Send Private SOL'
openapi: 'POST /privacy-cash/send'

Send SOL with complete privacy. Zero-trace offline transfers.

## Fees

**Total: 1% + 0.006 SOL**
- Privacy Cash: 0.35% + 0.006 SOL
- NovaShield: 0.65%

## Minimum

Recipient must receive at least **0.03 SOL** after fees. Send ~0.04 SOL minimum.

## Request

```json
{
  "fromPrivateKey": "[1,2,3,...]",
  "toAddress": "RECIPIENT_WALLET_ADDRESS",
  "amount": 0.05,
  "network": "mainnet-beta"
}
```

## Response

```json
{
  "success": true,
  "data": {
    "signature": "5J7x...",
    "amount": 50000000,
    "recipient": "RECIPIENT_WALLET_ADDRESS",
    "txId": "firebase-tracking-id"
  }
}
```

## How It Works

1. Auto-deposits to privacy pool
2. Processes transfer offline
3. Auto-withdraws to recipient
4. Keys saved to Firebase for recovery
